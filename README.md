# Python Mini-Projects

A collection of standalone Python projects built to practice and solidify foundational programming concepts.

---

## 01 - Command-Line Calculator

A simple calculator that performs basic arithmetic operations in the terminal.
-   **Concepts:** `input()`, `f-strings`, `if/elif/else`.

---

## 02 - Command-Line Bank

A text-based banking application to manage deposits, withdrawals, and balance.
-   **Concepts:** `functions`, `return values`, `while loops`, `state management`.
